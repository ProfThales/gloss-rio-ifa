Objetivo do Projeto

Este projeto apresenta práticas e tecnologias que tornam a web mais acessível para pessoas com baixa visão, contribuindo para a inclusão digital e para o desenvolvimento de interfaces mais humanas e eficientes.

O que é Baixa Visão?

A baixa visão é uma condição visual em que a pessoa apresenta redução significativa da capacidade visual, mesmo utilizando correção óptica convencional. Recursos digitais acessíveis podem ampliar significativamente a autonomia e a qualidade da experiência online desses usuários.

Boas Práticas de Desenvolvimento Web Acessível

  • Contraste adequado entre texto e fundo.
  • Suporte a zoom de até 200%.
  • Navegação completa utilizando teclado.
  • Indicadores de foco altamente visíveis.
  • Uso de HTML semântico.
  • Tipografia flexível utilizando unidades relativas.

Como adaptar um site para auxiliar pessoas com baixa visão

Sites acessíveis podem utilizar aumento de fonte, alto contraste, espaçamento ampliado, compatibilidade com leitores de tela e navegação por teclado. Essas medidas melhoram a legibilidade, a navegação e a experiência do usuário.

Glossário

Baixa Visão
Redução significativa da capacidade visual que não é totalmente corrigida por óculos ou lentes.
Contraste
Diferença visual entre elementos, como texto e fundo.
Reflow
Capacidade do layout se reorganizar quando o usuário utiliza zoom.
HTML Semântico
Uso correto das tags HTML para representar o significado do conteúdo.
WCAG
Diretrizes internacionais para acessibilidade na web.
